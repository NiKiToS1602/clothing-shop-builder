# Конструктор интернет-магазинов одежды

Платформа для создания и управления интернет-магазинами одежды.  
Проект включает backend API и два frontend-приложения: клиентскую витрину и административную панель.

Архитектура построена с расчётом на масштабирование, мульти-магазинность и чёткое разделение ответственности.

---

## Назначение проекта

Система позволяет:
- регистрировать и аутентифицировать пользователей по email (OTP-код);
- создавать и настраивать интернет-магазины одежды;
- управлять каталогом товаров (категории, товары);
- предоставлять SEO-оптимизированную клиентскую витрину;
- предоставлять PWA-админку для управления магазином и каталогом.

---

## Архитектура проекта

Проект организован как **monorepo** и включает backend и frontend части.

### Backend
Микросервисный backend с API-first подходом.

Сервисы:
- **auth** — аутентификация (email + OTP, JWT)
- **users** — пользователи, роли и права доступа *(планируется)*
- **store** — магазины и их настройки *(планируется)*
- **catalog** — категории и товары
- **content** — контент и страницы *(планируется)*

### Frontend
- **Client (витрина)** — клиентская часть интернет-магазина одежды
- **Admin (админка)** — административная панель управления магазином и каталогом
