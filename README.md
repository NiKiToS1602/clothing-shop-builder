# Конструктор интернет-магазинов канцелярии

Платформа для создания и управления интернет-магазинами канцелярских товаров.  
Проект включает backend API и два frontend-приложения: клиентскую витрину и административную панель.

Архитектура построена с расчётом на масштабирование, мульти-магазинность и чёткое разделение ответственности.

---

## Назначение проекта

Система позволяет:
- регистрировать и аутентифицировать пользователей по email (OTP-код);
- создавать и настраивать магазины;
- управлять каталогом канцелярских товаров;
- предоставлять SEO-оптимизированную клиентскую витрину;
- предоставлять PWA-админку для управления магазином.

---

## Архитектура проекта

Проект организован как **monorepo** и включает backend и frontend части.

### Backend
Микросервисный backend с API-first подходом.

Сервисы:
- **auth** — аутентификация (email + OTP, JWT)
- **users** — пользователи, роли и права доступа
- **store** — магазины и их настройки
- **catalog** — категории, бренды, размеры, товары
- **content** — новости и контент

### Frontend
- **Client (витрина)** — клиентская часть магазина
- **Admin (админка)** — управление магазином и каталогом
