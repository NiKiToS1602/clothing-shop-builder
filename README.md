# Конструктор интернет-магазинов канцелярии

Backend-платформа для создания и управления интернет-магазинами канцелярских товаров.  
Система поддерживает несколько магазинов, гибкую настройку каталога и разделение прав доступа.

Проект реализован на **Python** с использованием **микросервисной архитектуры**.

---

## Назначение проекта

Платформа позволяет:
- регистрировать пользователей через email (одноразовый код);
- создавать и управлять магазинами;
- настраивать каталог канцелярских товаров;
- предоставлять API для клиентской витрины и административной панели.

---

## Архитектура

Проект организован как **monorepo** и состоит из независимых backend-сервисов.

### Сервисы

| Сервис | Описание |
|------|---------|
| auth | Аутентификация по email (OTP), JWT-токены |
| users | Пользователи, роли и права доступа |
| store | Магазины и их настройки |
| catalog | Каталог товаров: категории, бренды, размеры, товары |
| content | Новости и контент |

---

## Технологический стек

- **Python 3.13**
- **FastAPI** — REST API
- **PostgreSQL** — основная база данных
- **Redis** — временные данные (OTP, кэш, лимиты)
- **SQLAlchemy 2.x** — ORM
- **Alembic** — миграции
- **Pydantic v2** — схемы данных
- **Docker / Docker Compose** — контейнеризация
- **pytest + pytest-cov** — тестирование
- **GitHub** — контроль версий
